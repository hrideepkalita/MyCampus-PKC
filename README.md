# ✨ About MyCampus

MyCampus is a modern campus platform built to improve student and campus interaction through technology. The project focuses on creating a smooth, secure, and user-friendly digital experience for students and administrators.

The platform is designed with scalability, usability, and modern web architecture in mind.

---

## 👨‍🎓 Student Side
